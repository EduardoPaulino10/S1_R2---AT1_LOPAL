Neste repositório contém exercícios realizados no Flowgorithm:

Idade:Este fluxograma pergunta a idade de uma pessoa e mostra se ela é maior ou menor de idade.

Nome:Este fluxograma pergunta o nome da pessoa que está digitando e exibe a seguinte mensagem: "Olá, (nome da pessoa)".

Nota:Este fluxograma solicita duas notas e calcula a média entre elas.

Positivo, Negativo e Zero: Este fluxograma analisa se o número é maior, igual ou menor que zero.

Revisão: Este fluxograma verifica se o número é maior ou menor que 10.

Soma: Este fluxograma realiza a soma de valores.
